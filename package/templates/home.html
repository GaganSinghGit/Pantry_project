Welcome to Pantry Master!


Deakin University's pantry database for Unit HSN754 - Dietary Modification to Improve Health



This website will allow students to access the available items in Deakin Univeristy's pantry and place requests to get these items.

Staff can view information about the food items available in the pantry, important information such as the requests placed and food stored in each container.
Staff also get an automatic shopping list created that list out the items that are in low supply and whose expiration dates are getting closer.


About Pantry Master



Pantry Master is a project developed for Unit SIT223 Professional Practice in IT.
This project has the following functionalities:

  1. Allow Students to Submit Request Forms for food items
  2. Shows current stock of items available in pantry
  3. Shopping list of items that need to be purchased

Navigating the website



Request

    Access the request form on this page.


Shopping List

    List of items that need to be purchased based on stock levels and expiry dates of each item.


Show Requests

    View all the request forms that have been submitted by students.


Containers

    Gives a complete overview of all the containers being used, the food items contained in the respective container and their barcodes.


Food Items

    Displays a table with all available food items and their current stock levels.
    Important information such as allergens, expiry dates, availability and its container ID can be viewed here.
